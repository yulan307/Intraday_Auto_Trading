# 开盘主导型三分类接口规格

## 1. 输入

```python
symbol: str
now_ts: datetime   # 当前时间，按 America/New_York 解释
```

---

## 2. 需要获取的数据列表

### 2.1 标的基础行情

必须获取：

1. 当日官方开盘价 `official_open`
2. 当前最新价 `last_price`
3. 当日实时 VWAP `session_vwap`
4. 当天 1 分钟 bar：

   * `open`
   * `high`
   * `low`
   * `close`
   * `volume`

### 2.2 开盘后 bar 数据范围

由 `now_ts` 决定：

* `09:30 <= now_ts < 09:35`
  获取 `09:30 ~ now_ts` 全部 1m bar

* `09:35 <= now_ts < 10:00`
  获取 `09:30 ~ now_ts` 全部 1m bar

* `now_ts >= 10:00`
  获取 `09:30 ~ 10:00` 全部 1m bar
  另外获取当前最新价 `last_price` 和当前 `session_vwap`

---

### 2.3 开盘撮合数据

如数据源支持，则获取：

1. `opening_imbalance_side`

   * `BUY`
   * `SELL`
   * `NONE`

2. `opening_imbalance_qty`

3. `paired_shares`

4. `indicative_open_price`

---

### 2.4 期权数据

选择规则：

1. 优先最近到期合约：

   * 优先 0DTE
   * 否则选择最近到期且 `DTE <= 7`

2. 以当前价为中心，获取：

   * ATM strike
   * ATM 下 1 档 strike
   * ATM 上 1 档 strike

3. 每个 strike 同时获取 call / put

每个合约需要字段：

* `bid`
* `ask`
* `bid_size`
* `ask_size`
* `last`
* `volume`
* `iv`（若有）
* `delta`（若有）
* `gamma`（若有）

---

### 2.5 期权快照时间点

至少获取两个时点：

1. `t_open_snapshot`
   开盘后首个有效快照

2. `t_now_snapshot`
   当前时刻快照

若 `now_ts >= 10:00`，则：

* `t_open_snapshot`
* `t_10m_snapshot`（约 09:40）
* `t_now_snapshot`

---

## 3. 返回预期结果

```python
{
    "symbol": str,
    "eval_time": str,
    "regime": "EARLY_BUY" | "RANGE_TRACK_15M" | "WEAK_TAIL"
}
```

---

## 4. 结果含义

* `EARLY_BUY`
  早买预期

* `RANGE_TRACK_15M`
  震荡预期

* `WEAK_TAIL`
  弱势拖尾预期
